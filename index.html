    The purpose of this project was to analyze the on-field statistics of Major League Baseball player and using the data to predict a ballplayer's salary for a given year. With the use of machine learning, I have created three models to predict salary, one by linear regression, one by convulutional neural network and another linear regression specifically designed to predict an outcome for Ronald Acuña Jr.
